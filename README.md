# spring-boot-demo

spring相关学习记录以及一些工具类



# 各模块介绍

## cloud

spring-cloud的学习模块,主要包括

1. 服务注册与发现
   - eureka
   - nacos
2. 服务负载与调用
   - ribbon
   - openFeign
3. 服务熔断与降级
   - hystrix
   - ~~sentinel(暂无研究)~~
4. 配置中心
   - nacos
5. 服务网关
   - gateway
